Basic concepts of HTML

Some Definitions

WWW
A collection of HTML documents.
HTML
They are the main type of documents on the web that glue everything together.
SERVERS
Computers that host files that make up the web.
HTTP
The main protocol of the web. When a person searches for a web page an HTTP request is sent to the web servers which will find the corresponding HTML document for the user.

Important ideas

Html is made of:

  • Text
  • Markup
    • This affects the way the text looks like. Some examples of tags are below:
    1. "b" is for bold
    2. "em" is for italics
    3. "br" is for break *doesn't require a closin tag
    4. "p" is for paragraphs
  • Reference to other files and links

Inline versus block elements:

  • Block tags create elements with invisible boxes around them. Some examples of block tags are "p" and "div" tags.
